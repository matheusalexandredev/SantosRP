# 🏙️ Santos RP - Landing Page

Esta é uma Landing Page moderna e de alta performance desenvolvida para o servidor de Roblox **Santos RP**. O projeto foi focado em criar uma experiência imersiva para o jogador, integrando dados em tempo real e uma interface visualmente impactante (Dark Mode).

---

## 🚀 Funcionalidades

- **Monitorização em Tempo Real**: Integração com a API do Roblox para exibir a contagem de jogadores ativos e visitas totais automaticamente.
- **Widget de Comunidade**: Exibição em tempo real dos membros online no Discord através do Widget oficial.
- **Navegação Fluida (UX)**: Implementação de *Smooth Scroll* para uma navegação contínua entre as secções (Sobre, Regras e Comunidade).
- **Design Responsivo**: Adaptado para uma visualização perfeita tanto em Desktop como em dispositivos móveis.
- **Performance**: Construído com código limpo (Vanilla JS/CSS), garantindo carregamento instantâneo.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica das secções.
- **CSS3**: Estilização avançada com Flexbox, Grid e efeitos de Glassmorphism no header.
- **JavaScript (ES6+)**: Consumo de APIs externas e manipulação do DOM.
- **Google Fonts**: Tipografia *Rajdhani* para um visual "Gaming/Tech".
